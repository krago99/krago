# Budget Klar

Eine klare Einnahmen/Ausgaben-App im Google-inspirierten Design.

Die App erfasst Einnahmen und Ausgaben, zeigt Beispiel-Daten wie Gehalt, Miete, Lebensmittel, Transport, Abos und Freizeit und bietet Monatsuebersicht, Diagramme, Suche, Filter und ein Formular fuer neue Buchungen.

Lokale Version: Die vollstaendige App wurde im Codex-Arbeitsordner `krago` erstellt.
